## Pontos de Discussão Importantes

### Contexto
Ambas trabalham com saúde e crianças com alterações emocionais via terapia ocupacional.

### 1. Cartão Digital
- Reunir informações gerais dos atendimentos
- Link direto para WhatsApp e redes sociais
- Design coerente com identidade visual (verde, tons terra, contexto saúde)

### 2. Site/Página de Vitrine
- SEO básico para ser encontrável no Google
- Responsivo (celular e computador)
- Conteúdo: apresentação das profissionais, serviços, grupos terapêuticos, contato
- Visual limpo, acolhedor e profissional
- Sem personalização pelo cliente (manutenção pela equipe)
- Possível expansão futura: cadastro de crianças, CRUD de turmas

### 3. WhatsApp
- Centralização de contatos via WhatsApp
- Link com mensagem padrão de interesse

### 4. Gestão de Turmas
- CRUD básico: nome, descrição, doença, responsável, telefone
- Visualizar alunos presentes, dia, hora, info básica

### 5. Identidade Visual
- Paleta: tons de verde e terra (calma, natureza, saúde)
- Logotipo profissional em busca de referências
- Padrão visual único para site e cartão
